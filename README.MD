# LeePathfindingGame

A simple game using Lee's algorithm.

## Controls:

- **Left Stick**: Moves the blue box across the grid.
- **B/Cross**: Places a green obstacle.
- **R/R1**: Starts the pursuit.
- **L/L1**: Clears all green obstacles.
- **Plus/Touchpad**: Exits the game.
